# My Machines

![Machines Overview](img/machines-0001.png)

The Machines module displays your complete inventory of installed Deckard machines. This comprehensive view helps you manage your machine fleet and quickly initiate inquiries for specific equipment.

## Machine List

### Overview

* View all machines in your installed base
* Each machine displays key information at a glance
* Filter and search capabilities for large fleets

### Updating Your Machine List

If your machine list is outdated or incomplete, please contact us to request an update. We'll ensure your records accurately reflect your current equipment inventory.

## Machine Details

![Machine Detail View](img/machines-0002.png)

Click on any machine in the list to open a detailed view in the right-hand panel.

### Information Displayed:

* Machine model and type
* Serial number
* Installation date
* Service history
* Technical specifications
* Compatible parts catalog

## Available Actions

From the machine detail view, you can perform the following actions:

### 1. Create Manual Inquiry

Initiate a new inquiry process for the selected machine.

**Process:**
* Click "Create Manual Inquiry" button
* System pre-fills machine information
* Enter part details and requirements
* Submit inquiry

See the [Manual Entry](#) chapter for detailed instructions on the inquiry process.

### 2. Download Machine Brochure

Access and download the technical brochure for the machine.

**Contains:**
* Technical specifications
* Part diagrams
* Maintenance guidelines
* Operating instructions
* Safety information

## Machine-Specific Benefits

### Pre-filled Information

When creating inquiries from the machine view:

* Machine model automatically selected
* Serial number included
* Installation date referenced
* Service history available for context

### Accurate Part Compatibility

The system ensures:

* Only compatible parts are recommended
* Correct specifications for your machine age
* Access to machine-specific catalogs

### Maintenance Insights

Track important information:

* Service due dates
* Part replacement history
* Recommended maintenance schedules
* Warranty information

---

**Best Practices:**
* Keep machine information current by reporting changes
* Review service history before ordering parts
* Download brochures for technical reference
* Use machine-specific inquiries for accurate part matching
* Contact support if you need to add new machines to your inventory
