# Customer Administration

## Home Dashboard

The Home Dashboard provides cex1ntralized access to all order management functions, organized into four main sections:

### History
View and manage all past orders, organized into three categories for easy navigation:
- **Latest**: Most recent completed orders
- **Completed**: All successfully fulfilled orders
- **Cancelled**: Orders that were cancelled

This categorization helps you quickly locate specific orders based on their outcome.

### Drafts
Access all orders currently in draft status. Draft orders are incomplete or pending finalization and can be edited or submitted when ready. Use this section to:
- Resume work on unfinished orders
- Review pending orders before submission
- Clean up unused draft orders

### Archive
Browse the complete list of archived orders. Archiving helps maintain a clean active workspace while preserving historical records for:
- Compliance and audit purposes
- Reference for future orders
- Long-term record keeping

### Bin
A temporary storage area for deleted orders, functioning as a safety net before permanent deletion. Key features:
- **Retention period**: 30 days
- **Recovery option**: Restore accidentally deleted orders within the retention period
- **Automatic cleanup**: Orders are permanently removed after 30 days

> **Tip:** Regularly review the Bin to recover any accidentally deleted orders before they're permanently removed.

## Machine Hub

### My Machines
Displays your complete inventory of installed machines. This centralized database provides:
- Quick access to all machine information
- Edit machine data or download machine brochuers and related documentation
- Overview of your equipment portfolio
- If a machine is not listed you can manually add a machine. All user added machines need to be approved by Deckard Administrator before they are visible in the machine list.

**Important:** If your machine list is outdated or incomplete, please contact us to request an update to ensure accurate service and support.

## User Hub

### Users
Manage all user accounts with full administrative control. This section allows administrators to:

#### User Management Actions
- **Add Users**: Create new user accounts for team members
- **Remove Users**: Delete user accounts that are no longer needed
- **Change Status**: Toggle users between "Active" and "Inactive" status

#### User Limitations
> **Important:** Only 5 active users are permitted at any given time. To add a new active user when at capacity, you must first deactivate an existing user.

#### Best Practices
- Deactivate users who are temporarily away rather than deleting their accounts
- Regularly review your active user list to ensure optimal use of available slots
- Keep user information current for security and communication purposes
