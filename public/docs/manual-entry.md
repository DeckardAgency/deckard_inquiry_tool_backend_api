# Manual Entry

Use Manual Entry when you have only partial information about the spare part or product, or when the item is not available in the standard shop catalog.

![Manual Entry Start](img/manual-0001.png)

## Getting Started

1. Start by selecting your machine from the list in the left column
2. **Note:** Only machines from your installed base are shown

![Machine Selection](img/manual-0002.png)

### Machine Search

You can filter the machine list by typing the machine name in the search field for faster selection.

![Required Data Entry](img/manual-0003.png)

After selecting your machine, you'll need to enter the required data for your inquiry.

## Two Entry Methods

![Entry Method Selection](img/manual-0004.png)

You can choose between two different methods in the upper right corner:

1. **Option "A"** - Use Input Form
2. **Option "B"** - Use Template

> **Important Warning:** Switching between "Input form" and "Use template" options will reset all unsaved data. You must complete your selection and submit or add items to cart before switching modes. A warning message will appear before any data is lost.

## Option "A" - Use Input Form

![Input Form](img/manual-0005.png)

The input form provides structured fields for detailed part information:

### Steps:

1. **Enter Required Data**
   - Part (product) name *
   - Part number (optional)
   - Short description *
   - Additional notes

2. **Upload Files**
   - Add images or documents that could help Deckard identify or understand your requirements
   - Supported formats: JPG, PNG, WEBP, PDF, DOC
   - Maximum file size: 5 MB per file

3. **Add Multiple Parts**
   - Click "Add part" button to include additional parts in the same inquiry
   - Each part can have its own files and notes

4. **Save or Continue**
   - **Save Draft** - Save your progress to continue later
   - **Continue** - Proceed to cart review and checkout

## Option "B" - Use Template

![Template Interface](img/manual-0006.png)

The template option provides a spreadsheet-like interface for bulk part entry:

### Features:

* **Spreadsheet Input**
  - Enter the data you know in the predefined sheet
  - Use familiar Excel-like interface

* **Demo Example**
  - Click on the "Demo example" tab at the bottom to view best practice approach
  - See how to properly fill out the template

* **Full Screen Mode**
  - Click on the "Expand button" (1) in the top right corner
  - Open the sheet in full screen for easier data entry

* **File Attachments**
  - Add additional files or images to support your inquiry

### Image Editor

When you click "Edit" next to an image file (.jpg/.png), a simple image editor appears:

![Image Editor](img/manual-0007.png)

**Editor Capabilities:**
* Write text directly on the image
* Make visual marks to highlight specific parts
* Circle or annotate areas that need replacement

## Cart Review

![Cart Sidebar](img/manual-0008.png)

When you add products to the cart, a sidebar with a list of added products appears.

### Available Actions:

* **Edit** - Modify part details
* **Remove** - Delete parts from the list
* **Continue** - Move to the next step in the checkout process

## Checkout

![Manual Entry Checkout](img/manual-0009.png)

The checkout page allows you to review all items before submitting your inquiry.

### Before Submitting:

* Review all products and details
* Remove any products you don't want to include
* **Important! Please enter "Internal reference number"!**

## Confirmation

After placing your inquiry, a thank you screen appears with options:

1. **Back to dashboard** - Return to main dashboard
2. **View active inquiries** - Go to "My inquiries" section
3. **Call / E-mail** - Contact your area manager

---

**Best Practices:**
* Provide as much detail as possible for faster processing
* Upload clear images showing the parts you need
* Use the image editor to highlight specific areas
* Include measurements or specifications when available
* Always add an internal reference number for tracking
